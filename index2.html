Introduction

PHP is a widely-used, open-source, server-side scripting language. It is designed primarily for web development but also serves as a general-purpose programming language. PHP scripts are executed on the server, generating HTML that is sent to the client, ensuring compatibility with any browser without relying on the client's capabilities. PHP is both powerful and flexible, making it ideal for building dynamic web pages and applications. It includes a comprehensive standard library with various built-in functions and classes to handle common tasks like file operations, database interactions, and session management. Core PHP can be extended and customized for various purposes by adding additional modules or using frameworks; for example:

  • Server-side PHP extends the core language by providing functionalities to manage server tasks. This includes database connectivity, session handling, file operations, and more. PHP supports numerous databases such as MySQL, PostgreSQL, and SQLite, enabling developers to create, read, update, and delete data with ease. Server-side PHP scripts can interact with databases using extensions like mysqli and PDO, manage sessions with session_start() and $_SESSION, and perform file manipulations using functions like fopen(), fread(), and fwrite().

  • Client-side PHP is not typical since PHP executes on the server. However, PHP can generate client-side code such as HTML, CSS, and JavaScript, which the browser then executes. This allows for the creation of dynamic content that can change based on server-side logic before being sent to the client.

Variables

In PHP, variables are used as symbolic names for values within an application. The names of these variables, referred to as identifiers, follow specific rules. A PHP identifier must start with a letter or an underscore (_); subsequent characters can be letters, underscores, or digits (0-9). PHP is case sensitive, so it distinguishes between uppercase and lowercase letters. Valid characters include "A" through "Z" (uppercase) and "a" through "z" (lowercase). Additionally, PHP allows the use of ISO 8859-1 or Unicode letters, such as å and ü, in identifiers. Some examples of legal PHP identifiers are $number_hits, $Temp99, and $_example.

If Else Statement

Use the if statement to execute a statement if a logical condition is true. Use the optional else clause to execute a statement if the condition is false. An if statement looks as follows:


if (condition) { statement_1; } else { statement_2; }


The condition can be any expression that evaluates to true or false. If the condition evaluates to true, statement_1 is executed; otherwise, statement_2 is executed. statement_1 and statement_2 can be any statement, including further nested if statements.


You may also compound the statements using else if to have multiple conditions tested in sequence, as follows:


if (condition_1) { statement_1; } else if (condition_2) { statement_2; } else if (condition_n) { statement_n; } else { statement_last; }


The condition can be any expression that evaluates to true or false. If condition_1 evaluates to true, statement_1 is executed; otherwise, if condition_2 evaluates to true, statement_2 is executed, and so on. If none of the conditions are true, statement_last is executed. statement_1, statement_2, statement_n, and statement_last can be any statement, including further nested if statements.


In the case of multiple conditions, only the first logical condition which evaluates to true will be executed. To execute multiple statements, group them within a block statement ({ ... }). In general, it's good practice to always use block statements, especially when nesting if statements:


if (condition) { statement_1_runs_if_condition_is_true; statement_2_runs_if_condition_is_true; } else { statement_3_runs_if_condition_is_false; statement_4_runs_if_condition_is_false; }


It is advisable not to use simple assignments in a conditional expression, because the assignment can be confused with equality when glancing over the code. For example, do not use the following code:


if ($x = $y) { /* statements here */ }



While Loop

A while statement executes its statements as long as a specified condition evaluates to true. A while statement looks as follows:


while (condition) statement


In PHP, if the condition becomes false, the statement within the loop stops executing and control passes to the statement following the loop. The condition is tested before the statement in the loop is executed. If the condition returns true, the statement is executed, and the condition is tested again. If the condition returns false, execution stops and control is passed to the statement following the while. To execute multiple statements, use a block statement ({ ... }) to group those statements.


In PHP, with each iteration of the while loop, the value of n is incremented and added to x. Therefore, x and n take on the following values:


  • Initially, n is 0 and x is 0.
  • In the first iteration, n is incremented to 1 and added to x, so x becomes 1.
  • In the second iteration, n is incremented to 2 and added to x, so x becomes 3.
  • In the third iteration, n is incremented to 3 and added to x, so x becomes 6.

After the third iteration, since n is no longer less than 3, the loop stops.
